# nixos-dotfiles

This repository contains one NixOS host:

- `scrappy`

## Entry points

- `flake.nix` wires NixOS and Home Manager together.
- `configuration.nix` imports the system modules.
- `home.nix` imports the user modules.
- `hardware-configuration.nix` is generated by NixOS and should usually stay machine-specific.

## System modules

System-wide settings live in `modules/nixos/`:

- `boot.nix`: bootloader
- `desktop.nix`: display manager, Niri, Firefox, PipeWire, fonts
- `hardware.nix`: firmware, keyboard module option, Broadcom Wi-Fi
- `networking.nix`: hostname and NetworkManager
- `nix.nix`: flakes and nixpkgs policy
- `packages.nix`: packages available for all users
- `power.nix`: power management, thermald, TLP
- `users.nix`: local user accounts

## Home Manager modules

User settings for `copperplate` live in `modules/home/`:

- `core.nix`: username, home directory, Home Manager state version, Git
- `cursor.nix`: cursor theme
- `packages.nix`: user packages
- `shell.nix`: Bash aliases and prompt
- `xdg-config.nix`: links `config/*` into `~/.config/*`

## Common commands

Check the config:

```sh
nix flake check --no-build
```

Apply the config:

```sh
sudo nixos-rebuild switch --flake ~/nixos-dotfiles#scrappy
```

Format Nix files:

```sh
alejandra .
```
